Average distance travelled*: .31 miles

*Statistic computed by performing the following steps:
  1. taking a random sample of the data (for cost and speed efficiency)
  2. using Google's Distance Matrix API to compute shortest distance between start and end of each point
  3. computing the haversine distance on these points
  4. determining the ratio between these two calculations, and
  5. applying this ratio to the average haversine distance of the whole dataset.
